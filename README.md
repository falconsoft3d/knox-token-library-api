# knox-token-library-api

# Dev 
```
git clone git@github.com:falconsoft3d/knox-token-library-api.git
cd knox-token-library-api
mv public.keys keys
npm i
npm run dev
```

# Run 
```
git clone git@github.com:falconsoft3d/knox-token-library-api.git
cd knox-token-library-api
mv public.keys keys
npm i
npm run start
```
